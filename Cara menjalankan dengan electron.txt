Cara menjalankan dengan electron
1. Install node.js
2. Download dan extract file ini
3. Buka terminal/command prompt
4. Jalankan 'npm init -y'
5. Jalankan 'npm install electron --save-dev'
6. Buat file main.js (jika belum ada)
7. Jalankan 'npm start'

Jika ingin menggunakannya lagi, cukup buka terminal/command prompt dan jalankan 'npm start
